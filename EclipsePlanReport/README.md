# EclipsePlanReport Source

Dies ist das C#-Projekt der Anwendung **Eclipse PlanReport**.

Build:

```powershell
dotnet build EclipsePlanReport.csproj -p:Configuration=Debug -p:Platform=x64
```

Die lauffaehige Ausgabe liegt danach in `debug\EclipsePlanReport.exe`.

Die fachliche Projektdokumentation liegt eine Ebene hoeher in `..\README.md`.
